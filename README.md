## How To Run
```
git clone https://github.com/Shaxadhere/universal-chakra-dashboard-boilerplate
cd universal-chakra-dashboard-boilerplate
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
